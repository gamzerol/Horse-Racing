# Horse Racing Game

Vue.js + Vuex horse racing simulation.
### Live Demo: https://horse-racing-orcin.vercel.app

## Features

- Random horse generation (1-20)
- 6 round race program
- Animated horse movement
- Vuex state management

## Tech Stack

- Vue 3
- Vuex
- TypeScript
- Vite

## Install

npm install

## Run

npm run dev
